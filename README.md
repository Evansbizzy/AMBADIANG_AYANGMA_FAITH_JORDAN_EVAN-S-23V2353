# TaxiData YDE

Une application web moderne pour la collecte et l'analyse des données de transport urbain à Yaoundé.

## Fonctionnalités
- **Collecte :** Formulaire par étapes (Stepper) pour l'enregistrement des courses.
- **Dashboard :** Visualisation interactive des données avec Plotly.
- **IA/ML :** Modèles de régression et classification avec Scikit-Learn.

## Stack Technique
- **Frontend/Backend :** Streamlit (Python)
- **Base de données :** Supabase (PostgreSQL)
- **Modèles :** Scikit-Learn

## Installation
1. `pip install -r requirements.txt`
2. `streamlit run app.py`
